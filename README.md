# 🔑 Automated Cryptographic Cipher Tool

## What it Does
This Python script implements a Caesar Cipher encryption and decryption algorithm. It starts by taking raw text inputs and applies algorithmic shifting to scramble up the data into ciphertext, demonstrating the foundational mechanics of data-at-rest encryption and symmetric key cryptography.

## Computer Science Concepts Applied
* **Character Encoding**: Utilizing Python's build-in 'ord()' and 'chr()' functions to manipulate raw ASCII/Unicode byte numerical data points of characters.
* **String Iteration**: Using loops to sequentially process, evaluate, and transform character arrays.
* **Modular Arithmetic**: Applying the modulo operator (' % 26') to wrap letter transformations correctly within the bounds of the English alphabet.
